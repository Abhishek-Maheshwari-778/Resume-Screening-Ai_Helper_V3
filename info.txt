--- 🛠️ PROTOCOL DEPLOYMENT (HOW TO RUN) ---

1. BACKEND (FastAPI - Neural Hub):
   - CD into 'backend' node.
   - Activate environment: `.\venv\Scripts\activate`
   - Install Neural dependencies: `pip install -r requirements.txt`
   - Sync spaCy model: `python -m spacy download en_core_web_sm`
   - Deploy server: `uvicorn main:app --reload --port 8000`

2. FRONTEND (Vite/React - Interface DNA):
   - CD into 'frontend' node.
   - Initialize modules: `npm install`
   - Deploy UI: `npm run dev` (http://localhost:5173)

--- 🗺️ SYSTEM ARCHITECTURE (V3) ---

- **Neural Synchronization**: Frontend communicates via the `/api/` gateway to the FastAPI LPU hub.
- **Groq LPU Engine**: Powering all structural summaries, resume deconstruction, and JD synthesis.
- **NLP Matrix**: Leveraging spaCy NER for sub-second entity extraction and skill mapping.
- **HR JD Synthesizer**: Specialized node for HR to generate & refine requirements via AI dialogue.
